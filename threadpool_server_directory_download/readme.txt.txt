Threadpool server that uses an executor service and allows up to a maximum of 10 clients to connect at once and logs all client requests in a .txt file.

Coursework 2 of Networks module within University of Leeds, Computer Science.

commands:

Syntax:
view *filepath*
Example:
view /folder1/images
Description:
This command allows the user to view all contents within the servers directory.

Syntax:
dl *filepath*
Example:
dl /folder1/images/img1.jpg
Description:
downloads the individual file from servers directory to the clients directory.

Syntax:
dirdl *filepath*
Example:
dirdl /img/folder1
Description:
downloads a directory of contents from the servers directory to the clients directory.
